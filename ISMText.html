ISMText: Wireless Messaging using LoRa Radio Modules

Project Description

For my final project in ECE 3140: Embedded Systems, my partners and I designed a wireless messaging system using LoRa radio modules integrated with NXP FRDM development boards. The system allows for long-range text communication via the terminal on two users' computers.

The system uses two FRDM boards, each connected to a LoRa module and a computer via USB. The FRDM boards are programmed in C using the MCUXpresso IDE to handle serial communication with the computer and wireless communication with the LoRa modules. The LoRa modules operate in the 915 MHz ISM band, theoretically allowing for long-range communication up to several kilometers.

Out of 99 projects in the class, our project was voted by our peers as the 7th best overall. The website for our project can be found here.
