Thruster Board

Project Description

Thruster Board was my first custom PCB, which I designed during my freshman year for Cornell's Autonomous Underwater Vehicle (AUV) Project Team. Thrusters board utilizes 4 electronic speed converters (ESCs) to control each of the submarine's underwater thrusters. The job of the board is to provide each ESC with 16V at up to 24A and a PWM signal to control the speed of that thruster. Two boards are used on the submarine to control a total of 8 thrusters.

Project Details

  • Project Name: Thrusters Board
  • Project Type: Schematic Capture, PCB Layout, Population, and Testing.
  • Project Timeline: October 2023 - April 2024
  • Tools Used: KiCad, C Programming
